// What I've Built

Projects

Heat Street

Group Project · Java

A competitive card game where 2–4 players are rival thieves racing to steal a Loot card from the center of a 7×7 board and escape. Players scatter traps to block opponents while using Utility cards to clear their own path. A Heat Meter tracks how "hot" each player is — reach 5/5 and you're caught, losing your loot and getting sent back to start.

Java OOP Game Design Playtesting Excel / Spreadsheet Design
